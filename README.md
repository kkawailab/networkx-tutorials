# NetworkX チュートリアル

NetworkXを使ったグラフ分析の包括的なチュートリアルです。初心者から上級者まで、段階的に学習できる構成になっています。

## 概要

このチュートリアルは、Pythonのグラフ分析ライブラリ[NetworkX](https://networkx.org/)を使って、グラフ理論の基礎から高度なネットワーク分析までを学べる教材です。

### 対象者

- グラフ理論・ネットワーク分析を始めたい方
- NetworkXの使い方を体系的に学びたい方
- ソーシャルネットワーク分析、生物ネットワーク、交通ネットワークなどの分析に興味がある方

## 使い方

### 必要条件

```bash
pip install networkx matplotlib numpy pandas scipy japanize-matplotlib
```

### 実行方法

1. このリポジトリをクローンまたはダウンロード
2. Jupyter NotebookまたはJupyter Labを起動
3. 各ノートブックを順番に実行

```bash
jupyter notebook
# または
jupyter lab
```

## 学習の進め方

### 推奨する学習順序

1. **初心者の方**: 第1章から順番に進めてください
2. **Python経験者**: 第1章を軽く確認後、第2章から開始
3. **グラフ理論経験者**: 興味のある章から始めてOK

### 各章の構成

- **学習目標**: 章で学ぶ内容
- **解説**: 概念と機能の説明
- **例題**: 実践的なコード例
- **練習問題**: 理解を深めるための課題（解答付き）
- **章のまとめ**: 重要ポイントの復習

## 特徴

- 豊富な例題と練習問題（各章3問以上）
- すべての練習問題に模範解答付き
- 実践的なユースケース（SNS分析、交通ネットワーク等）
- グラフの可視化を重視
- 日本語での丁寧な解説

## 主なトピック

- グラフの作成と操作
- グラフアルゴリズム（最短経路、中心性、クラスタリング等）
- コミュニティ検出
- ネットワーク可視化
- 実データの分析
